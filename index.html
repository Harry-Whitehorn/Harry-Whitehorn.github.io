Bootcamp 2021: Conway's Game of Life

My first programming project, concluding the 2 week Programming Bootcamp at City (2021). This Processing project demonstrates a comprehensive understanding of the use of 2d arrays, loops and reading and wring to files. I created a 2d boolean grid and store it in a 2d array. The user is able to click to toggle a grid cell on or off. Then, while the program is running, each cell is updated based on its number of neighbors.

The Rules of Life

  • Underpopulation- If an alive cell has fewer than two alive neighbors, it dies.
  • Idle - If an alive cell has two or three alive neighbors, it stays alive.
  • Overpopulation - If an alive cell has greater than three alive neighbors, it dies.
  • Reproduction - If a dead cell has exactly three alive neighbors, it becomes alive.
